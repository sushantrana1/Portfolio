# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sushantrana1/your-portfolio-repo.git
```

Go into the project folder

```bash
cd your-portfolio-repo
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
